# KStory (inDev)

## Overview

KStory is **visual novel**, **point-and-click exploration**, and **mini-games work simulator**.

<img width="1919" height="1079" alt="изображение" src="https://github.com/user-attachments/assets/accf36c1-eb09-4b32-a7a8-ceabda5fc554" />


In this **game You'll**:
- Navigate the village using an interactive map
- Talk to eccentric locals through dialogue choices
- Inspect environments for clues
- Complete paperwork mini-games to progress

### Mini-games
  - *Kraepelin Test* timed arithmetic calculations
  - *Documentation work* filling in the blanks on the form
  - *Schulte Tables* sequential number finding

### Technical Implementation
- Built with **Ren'Py Engine**
- Python-based mini-game systems
- Custom UI screens
